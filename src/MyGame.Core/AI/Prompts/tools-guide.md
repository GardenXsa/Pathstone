# Гид по инструментам

> **Примечание порта (C#/.NET 8).** В исходном TypeScript-проекте этот файл
> был пуст — полный набор инструментов реализован в `src/ai/tools/index.ts`
> (2.5+ тыс. строк). В C#-порту набор инструментов намеренно сокращён до
> стартового (`roll_dice`, `get_player_state`, `get_location`, `spawn_npc`,
> `advance_time`) — см. `MyGame.Core.AI.Tools.ToolRegistry`. Полный набор
> будет перенесён отдельной задачей. Ниже — краткая справка по стартовым
> инструментам.

## Стартовый набор

### roll_dice
Бросок кубиков по формуле `NdM±K` (напр. `2d6+3`, `1d20`, `3d8-1`).
Поля: `expression` (string, обязательный), `purpose` (string, опц.).

### get_player_state
Возвращает снимок состояния активного игрока: имя, раса, класс, уровень,
характеристики, ресурсы, экипировка, инвентарь, локация. Полей нет.

### get_location
Возвращает описание локации по ID или имени: terrain, danger, выходы,
обитатели (NPC), здания, предметы на земле. Поле: `locationId` (string,
опц. — по умолчанию текущая локация игрока).

### spawn_npc
Заспавнить NPC из реестра шаблонов в локации. Поля: `templateId` (string,
обязательный), `locationId` (string, опц. — текущая локация игрока),
`nameOverride` (string, опц.).

### advance_time
Продвинуть внутриигровое время. Поле: `minutes` (int, по умолчанию 0).
Возвращает новое значение часов мира.

## Правила использования

- Любое механическое воздействие (бросок, спавн, перемещение времени)
  делается ТОЛЬКО через инструменты, не текстом.
- Сначала опиши действие в наррации, затем вызови инструмент, затем
  опиши результат.
- Если инструмент вернул ошибку — поправь аргументы и попробуй снова.
